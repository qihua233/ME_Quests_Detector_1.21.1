# ME_Quests_Detector_1.21.1

基于原项目 ME_Quests_Detector 的 NeoForge 1.21.1 适配版本。

## 当前状态

- 目标平台：Minecraft 1.21.1
- 加载器：NeoForge 21.1.220
- AE2：19.2.17
- FTB Quests：2101.1.24
- FTB Teams：2101.1.10
- FTB Library：2101.1.31

## 项目说明

该模组用于将 AE2 存储网络与 FTB Quests 的提交/检测任务联动，支持物品与流体任务检测。

当前仓库基于原作者 LuckyBlock、mmyddd 的项目继续适配与修复。
项目使用了AI协助。
